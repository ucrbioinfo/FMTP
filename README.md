Download: type "git clone https://github.com/ucrbioinfo/FMTP"

Compite: type "cd FMTP/source" then modify the the path of the Boost C++ library in Makefile-MODIFY_BEFORE_USING.
Change the filename of Makefile-MODIFY_BEFORE_USING to Makefile then type "make" to compile FMTP.

To run FMTP, enter run_dir directory and type "./run_script_FMTP" to run FMTP on the sample data set. You could change the parameter in the script files to run it on another data set.
